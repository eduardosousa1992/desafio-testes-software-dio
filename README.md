# 🚀 Planejamento de Testes e Gestão de Qualidade

Este repositório contém a documentação técnica desenvolvida para o desafio de projeto sobre o ciclo de vida de software e processos de teste da DIO.

## 📋 Conteúdo do Projeto

O arquivo PDF principal inclui:

1.  **Plano de Fluxo de Trabalho (Workflow):** Definição das etapas de desenvolvimento.
2.  **Ciclo de Vida do Bug:** Mapeamento dos estados de um defeito.
3.  **User Stories:** Descrição de funcionalidades focadas no usuário.
4.  **Documentos de Teste:**
    * **Mind-map:** Visão holística dos cenários.
    * **Testes Step-by-Step:** Casos de teste detalhados.
    * **Testes em BDD (Gherkin):** Cenários no formato *Dado, Quando, Então*.
