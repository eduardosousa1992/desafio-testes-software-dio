# 🚀 Planejamento de Testes e Gestão de Qualidade

Este repositório contém a documentação técnica desenvolvida para o desafio de projeto sobre o ciclo de vida de software e processos de teste. O objetivo é demonstrar a estruturação de fluxos de trabalho, gestão de defeitos e escrita de requisitos sob a ótica de QA.

## 📋 Conteúdo do Projeto

O arquivo PDF principal inclui:

1.  **Plano de Fluxo de Trabalho (Workflow):** Definição das etapas de desenvolvimento (To Do, In Progress, Testing, Done).
2.  **Ciclo de Vida do Bug:** Mapeamento dos estados de um defeito, desde a identificação até o fechamento.
3.  **User Stories:** Descrição de funcionalidades focadas no valor de negócio e experiência do usuário.
4.  **Documentos de Teste:**
    * **Mind-map:** Visão holística dos cenários de teste.
    * **Testes Step-by-Step:** Casos de teste detalhados com passos e resultados esperados.
    * **Testes em BDD (Gherkin):** Cenários descritos no formato *Dado, Quando, Então*.
